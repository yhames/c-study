## 전처리기

`전처리기`는 `소스코드`를 `변환단위`로 수정하여 `컴파일러`에게 전달한다.

전처리기는 문법 검사를 하지 않는다.
전처리기는 오직 텍스트 치환을 위주로 하는 등의 간단한 작업만 수행한다.

전처라기는 별도의 구문분석을 위한 파서(`parser`)를 사용한다.
이는 `C`언어와는 독립적이기 때문에 다른 용도로도 사용할 수 있다.
`UNIX` 계열에서는 전처리기로 `cpp`을 사용한다(이는 `C++`와는 관련이 없다.)

다은과 같은 방식으로 `cpp`를 사용할 수 있다.
```bash
cpp ExtremeC_examples_chapter2_1.C
```

마지막으로 `변환단위`는 `.i` 확장자를 갖는다.
따라서 `.i` 확장자를 갖는 코드를 전처리하면 이미 전처리가 완료되었다는 `warning`을 출력한다.
